# OfferOptimizer

OfferOptimizer is a machine learning solution built during the HCLTech Hackathon @ NSUT to predict the optimal discount a customer is likely to accept based on their past purchases and loyalty tier. This helps businesses optimize promotional spending, personalize offers, and improve customer engagement.  
The model achieved a **25% improvement in recall** compared to the baseline.
